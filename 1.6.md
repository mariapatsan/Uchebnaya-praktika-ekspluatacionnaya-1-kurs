## Механизм автоматизированной установки Ubuntu:
В Ubuntu существует механизм автоматизированной установки Autoinstall, основанный на технологии Cloud-Init. Данный механизм позволяет установить операционную систему без участия пользователя.
Для автоматической установки создаётся файл конфигурации user-data, в котором задаются параметры будущей системы:
•	имя компьютера;
•	имя пользователя;
•	пароль;
•	язык системы;
•	раскладка клавиатуры;
•	параметры сети;
•	разметка диска;
•	список дополнительных пакетов.

Пример конфигурации:
autoinstall:
  version: 1
  identity:
    hostname: ubuntu-vm
    username: student
    password: password
После запуска установщика Ubuntu параметры считываются автоматически, и система устанавливается без ручного ввода данных.
Автоматизированная установка прикладного ПО:
В качестве прикладного программного обеспечения выбран офисный пакет LibreOffice.
Для автоматизации установки используется Bash-скрипт.

## Скрипт установки LibreOffice:

Файл install_libreoffice.sh:
#!/bin/bash

echo "Обновление списка пакетов..."
sudo apt update

echo "Установка LibreOffice..."
sudo apt install -y libreoffice

echo "Установка завершена."

Назначение команд:
apt update - обновление списка доступных пакетов;
apt install libreoffice - установка LibreOffice;
ключ -y автоматически подтверждает установку без участия пользователя.

## Алгоритм установки LibreOffice:
1.	Запустить Ubuntu Linux.
2.	Открыть терминал.
3.	Создать файл install_libreoffice.sh.
4.	Вставить в него текст скрипта.
5.	Сохранить файл.
6.	Назначить права на выполнение:
chmod +x install_libreoffice.sh
7.	Выполнить запуск скрипта:
./install_libreoffice.sh
8.	Дождаться завершения установки.
9.	Проверить наличие LibreOffice в меню приложений.
10.	Запустить программу и убедиться в её работоспособности.
