# Основные стандарты и спецификации в сфере информационных технологий

1. ГОСТ Р ИСО/МЭК 12207–2010. Процессы жизненного цикла программного обеспечения — Текст: электронный // rbsnt-new.ru: [сайт]. — URL: https://www.rbsnt-new.ru/wp-content/uploads/2020/11/%D0%93%D0%9E%D0%A1%D0%A2-12207-2010.pdf (дата обращения: 04.06.2026).
Определяет процессы разработки, сопровождения и эксплуатации программного обеспечения на всех этапах его жизненного цикла. Является базовым стандартом для построения систем менеджмента качества в ИТ-компаниях.

2. ГОСТ Р ИСО/МЭК 25010-2015. Модель качества программного обеспечения — Текст: электронный // internet-law.ru: [сайт]. — URL: https://internet-law.ru/gosts/gost/60038/ (дата обращения: 04.06.2026).
Устанавливает характеристики качества ПО: функциональность, надёжность, безопасность, производительность и удобство использования. Включает восемь основных характеристик и их подхарактеристики (тестируемость, адаптируемость, восстанавливаемость).

3. ISO/IEC 27001. Система менеджмента информационной безопасности. — Текст: электронный // clck.ru: [сайт]. — URL: https://clck.ru/3TzM6r (дата обращения: 04.06.2026).
Определяет требования к защите информации и управлению рисками информационной безопасности. Является основой для сертификации организаций на соответствие стандартам информационной безопасности.

4. ISO/IEC 20000. Управление ИТ-услугами. — Текст: электронный // meganorm.ru: [сайт]. — URL: https://meganorm.ru/Data2/1/4293747/4293747574.pdf (дата обращения: 04.06.2026).
Содержит требования к организации и управлению ИТ-сервисами в компании. Гармонизирован с методологией ITIL (Information Technology Infrastructure Library).

5. IEEE 802.3 (Ethernet). — Текст: электронный // repka-pi.ru: [сайт]. — URL: https://repka-pi.ru/docs/100 (дата обращения: 04.06.2026).
Аннотация: Основной стандарт проводных локальных сетей, определяющий принципы передачи данных по технологии Ethernet. Регулирует физический уровень и уровень канала передачи данных.

6. IEEE 802.11 (Wi-Fi). — Текст: электронный // ru.ruwiki.ru: [сайт]. — URL: https://ru.ruwiki.ru/wiki/IEEE_802.11 (дата обращения: 04.06.2026).
Стандарт беспроводных локальных сетей, используемый для организации Wi-Fi-соединений. Включает множество поправок (a/b/g/n/ac/ax), определяющих скорость, частотные диапазоны и методы шифрования.

7. TCP/IP (RFC 791 и RFC 793). — Текст: электронный // OpenNet: [сайт]. — URL: https://www.opennet.ru/docs/RUS/tcpip/ (дата обращения: 04.06.2026).
Базовый набор сетевых протоколов Интернета, обеспечивающий обмен данными между устройствами. Включает протокол IP (адресация и маршрутизация) и TCP (надёжная передача данных с установлением соединения).

8. HTML (HTML Living Standard / W3C Recommendation). — Текст: электронный // sky.pro: [сайт]. — URL: https://sky.pro/wiki/javascript/w3c-i-veb-razrabotka-standarty-html5-dostupnost/ (дата обращения: 04.06.2026).
Стандартный язык разметки для создания и отображения веб-страниц. Определяет структуру документа с помощью тегов и элементов. Актуальная версия - HTML5.

9. CSS (CSS Level 3+ / W3C Recommendation). — Текст: электронный // ruwiki.ru: [сайт]. — URL: https://ru.ruwiki.ru/wiki/CSS (дата обращения: 04.06.2026).
Спецификация для оформления и визуального представления веб-документов. Позволяет отделять содержимое (HTML) от его отображения (цвета, шрифты, расположение элементов).

10. JavaScript (ECMAScript, стандарт ECMA-262). — Текст: электронный // ru.ruwiki.ru: [сайт]. — URL: https://ru.ruwiki.ru/wiki/ECMAScript (дата обращения: 04.06.2026).
Стандарт языка программирования, используемого для создания интерактивных веб-приложений. Определяет синтаксис, типы данных, объекты и механизмы выполнения кода. Актуальные версии - ES2020 и новее.

11. ГОСТ 34.601–90. — Текст: электронный // normativ.kontur.ru: [сайт]. — URL: https://normativ.kontur.ru/document?moduleId=9&documentId=57693 (дата обращения: 04.06.2026).
Устанавливает стадии создания автоматизированных систем от разработки до ввода в эксплуатацию. 

12. ГОСТ 34.602–89. — Текст: электронный // normativ.kontur.ru: [сайт]. — URL: https://normativ.kontur.ru/document?moduleId=9&documentId=200428 (дата обращения: 04.06.2026).
Определяет требования к техническому заданию на разработку автоматизированной системы. Устанавливает состав, содержание и оформление документа «Техническое задание».


13. ISO/IEC 29119 (серия). Тестирование программного обеспечения. — Текст: электронный // ohranatruda.ru: [сайт]. — URL: https://ohranatruda.ru/upload/iblock/686/4293754867.pdf (дата обращения: 04.06.2026).
Международный стандарт, охватывающий процессы, документацию и методы тестирования ПО. Заменил устаревший IEEE 829. Определяет единую терминологию и подходы к тестированию на всех этапах разработки.
