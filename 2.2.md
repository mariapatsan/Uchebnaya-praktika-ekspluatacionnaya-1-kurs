# Прикладное программное обеспечение информационно-вычислительной системы образовательной организации

## 1. Решаемые задачи, связанные с автоматизированной обработкой информации

Автоматизированное рабочее место преподавателя информатики в аудитории 323 предназначено для решения следующих задач с помощью автоматизированной обработки информации:

|Задача	|Описание|
|-------|-----|
|Подготовка учебно-методических материалов	|Создание текстовых документов, презентаций, таблиц, раздаточного материала|
|Обучение программированию	|Написание, отладка и выполнение программ на языках Python, C++, Pascal|
|Обучение работе с базами данных	|Изучение структуры БД, выполнение запросов, работа с СУБД Firebird и 1С|
|Математические и инженерные расчёты	|Вычисления в системах компьютерной алгебры Maxima и Scilab|
|Демонстрация учебного материала	|Вывод изображения на четыре монитора через HDMI Splitter|
|Организация дистанционного обучения	|Электронная почта (Outlook), облачное хранение (OneDrive), веб-доступ|
|Ведение документации преподавателя	|Журналы, отчёты, планы занятий (MS Office)|
|Обеспечение информационной безопасности	|Антивирусная защита, контроль сети (Kaspersky)|
|Администрирование и мониторинг ПК	|Диагностика оборудования (PCInfo), системные утилиты|
|Виртуализация	|Запуск гостевых операционных систем для обучения (Oracle VM VirtualBox)|
